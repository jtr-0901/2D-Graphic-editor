# 2D Graphics Editor in C

A simple menu-driven 2D graphics editor developed in C using a 2D character array.

## Features

* Draw Line
* Draw Rectangle
* Draw Triangle
* Draw Circle
* Add, Delete and Modify Shapes
* Display Canvas

## Files

* `main.c` - Menu and user interaction
* `canvas.c/h` - Drawing functions and canvas handling
* `shapes.c/h` - Shape management

## Compile

```bash
gcc main.c canvas.c shapes.c -o editor.exe

## Run

```bash
editor.exe
